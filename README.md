# Delta Ops: Hazard Zone (核心防護與經濟系統)

本模組為 《Delta Ops: Hazard Zone》 的核心安全與經濟管理系統，具備雙重遠端代碼完整性校驗與 HMAC 設定檔動態簽署機制。

## 🛡️ 安全功能
- **代碼完整性盾牌**：連線至 Cloudflare Worker 進行 SHA-256 指紋即時校驗，嚴防非法篡改。
- **經濟防護網**：當代碼遭篡改時，全面凍結經濟結算系統並清空內存數據。
- **安全診斷系統**：支援管理員免 OP 雙通道驗證，即時導出伺服器安全報告。

## 🛠️ 管理員指令
- `/dt diag`：導出目前伺服器的完整安全與數據診斷報告（需 OP Level 2 或名單內 UUID）。
- `/dt config sign <檔案名>`：對物價設定檔進行 HMAC 安全簽署，並全服廣播同步封包。

---

這是一個基於 **Minecraft Forge 1.20.1** 的戰術撤離射擊風格模組（Tarkov-like），實作了完整的撤離、經濟、安全箱、小隊、戰利品容器、部位傷害等核心玩法。

## 🎯 核心玩法

### 1. 撤離系統
- 玩家進入撤離區域後自動啟動 **10 秒倒數**
- 撤離完成後結算背包物品總價值 → 加入哈夫幣餘額 → 傳送回大廳
- 支援伺服器拉閘撤離（15 秒倒數，180 秒閘門計時）
- 每張地圖可設定多個撤離點
- 管理員可透過 GUI 或 `/dt extract add <mapName>` 設定撤離點

### 2. 經濟系統
- 內建 **哈夫幣（Haf币）** 餘額系統（儲存於玩家 PersistentData）
- 物品價格儲存於 `item_prices.json`，支援動態修改與重載
- 撤離時計算背包所有物品總價值 × 1.5 倍 + 地圖基礎獎勵金
- 管理員可透過 GUI 或 `/dt balance` 查詢餘額

### 3. 安全箱系統
- 安全箱等級 **1-4 級**，格子數：2 / 4 / 6 / 9
- 安全箱內的物品 **死亡後完全保留**，不會掉落
- 按 **O 鍵** 快速開啟安全箱
- **安全箱需要購買才能使用**：前往商人商店購買安全箱解鎖（Lv1: 10,000 / Lv2: 50,000 / Lv3: 150,000 / Lv4: 500,000 哈夫幣）
- 安全箱資料儲存於玩家 PersistentData，死亡後保留

### 4. 死亡懲罰系統
- 玩家死亡時：取消原版掉落物
- 手動將主背包、護甲欄、副手全部丟出
- 安全箱內物品不受影響
- 對戰中死亡 → 自動切換為旁觀者模式

#### 觀戰模式操作
| 操作 | 說明 |
|------|------|
| 數字鍵 **1** | 退出觀戰，返回大廳 |
| 數字鍵 **2~9** | 切換跟隨不同存活隊友的視角 |
| 自動鎖定 | 鏡頭會鎖定在隊友身上，無法自由移動看到敵方位置 |
| 全隊陣亡 | 自動返回大廳 |

> 觀戰者只能看到同小隊存活隊友的視角，**無法切換到敵方玩家**，防止報點作弊。

### 5. 管理員設定介面
- 開啟方式：`/dt admin config`
- **完整功能：**
  - 切換選擇地圖
  - 設置當前選擇地圖的撤離點
  - 重載物價表
  - 修改物品價格
  - **安全箱等級設定**（1~4 級循環切換）
  - **死亡掉落規則**（全部掉落 / 僅背包 / 不掉落）
  - **撤離倒數時間**（5 / 10 / 15 / 20 / 30 / 60 / 120 秒）
  - **地圖獎勵倍率**（0.5x / 1.0x / 1.5x / 2.0x / 3.0x / 5.0x / 10.0x）

|----------|------|------|------|------|------|------|

| LOW_TIER | - | - | 80% | 19% | 1% | - |
| TRASH_TIER | 70% | 30% | - | - | - | - |

### 容器生成機制
- **固定錨點（Fixed Anchors）**：每張地圖可註冊固定高價值容器位置
- **網格分佈（Grid Distribution）**：將對戰區域劃分網格，每個網格隨機放置容器
- **每局重置**：發車時自動清除所有舊容器 → 重新生成（固定錨點 + 隨機分佈）

### 物品分類（Loot Categories）
`COLLECTIBLE` · `ELECTRONIC` · `KEYCARD` · `WEAPON` · `MEDICAL` · `AMMO`

### 管理員戰利品標籤工具
- 開啟方式：`/dt admin items`
- 手持物品後選擇類別與品質
- 儲存後該物品會按對應分類和品質出現在戰利品生成池中

## 🔄 配對系統

### 配對流程
1. 玩家透過 `/dt menu` → **配對大廳** 進入配對畫面
2. 點擊 **開始配對** → 加入佇列（單人或小隊）
3. 系統自動尋找其他佇列玩家補人
4. 配對成功 → 隊長按 **發車**
5. 發車後自動執行：
   - ✅ 戰備值檢查
   - ✅ 清除範圍內所有舊容器
   - ✅ 生成固定高價值容器
   - ✅ 網格隨機分佈戰利品容器
   - ✅ 傳送至隨機重生點
   - ✅ 標記為對戰玩家（死亡切旁觀）
