# 🛠️ 支援指南 (Support Guidelines)

歡迎來到 **Delta Ops: Hazard Zone (Alpha)**！
如果您在遊玩或測試過程中遇到任何問題、需要協助，或是想提供建議，請透過以下官方管道聯繫我們。

---

## 1. 💬 Discord 官方社群 (首選推薦 / Recommended)

我們強烈建議您加入官方 Discord 伺服器，這裡是我們處理技術支援與社群交流的核心樞紐。

* **加入連結**：[點擊加入 Delta Ops Discord](https://discord.gg/ZDqB76naFu)
* **如何尋求協助？**
  進入伺服器後，請前往客服支援頻道，點擊 **Ticket Tool (票務工具)** 按鈕。系統會為您開啟一個專屬的私人對話框，您可以選擇以下服務：
  * **一般客服 (General Support)**：遊戲疑問、安裝教學等。
  * **一般問題回報 (General Issues)**：非緊急的 Bug 或功能建議。
  * **緊急客服/問題回報 (Urgent Support/Issues)**：導致遊戲崩潰、伺服器無法運作的嚴重錯誤。
  
  *(支援中英雙語溝通 / Bilingual support available)*

---

## 2. 🐛 GitHub Issues (技術與 Bug 追蹤)

如果您確定發現了一個遊戲 Bug，或者有具體的程式碼/模組功能建議，請直接在 GitHub 上提交 Issue。這有助於我們系統化地追蹤與修復。

* **前往回報**：[GitHub Issues](https://github.com/Ray20123315/delta_ops_hazardzone_Alpha/issues)
* **回報要求**：請盡量提供完整的錯誤日誌 (Crash Logs) 與觸發 Bug 的具體步驟。

---
