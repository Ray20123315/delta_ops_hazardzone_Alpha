# 行為準則 (Code of Conduct)

## 我們的承諾

為了建立一個開放與友善的環境，我們承諾尊重所有參與 **Delta Ops: Hazard Zone** 專案的成員。無論您是透過 GitHub 提交 Issue、在 Discord 參與討論，或是加入 QQ 群交流，我們都期望大家能共同維護一個無騷擾、互相尊重的社群。

## 我們的標準

**我們鼓勵以下有助於建立正面社群環境的行為：**
* 使用友善、包容的語言進行交流。
* 尊重不同的遊戲體驗與測試反饋。
* 友善地接受建設性的批評與建議。
* 聚焦於對專案與社群整體有益的事物。

**我們嚴格禁止以下行為（包含但不限於）：**
* 發表冒犯性、侮辱性、歧視性或貶低他人的言論。
* 公開或私下騷擾社群成員。
* 惡意洗版、發送垃圾訊息 (Spamming) 或未經授權的廣告宣傳。
* 惡意利用 Bug 破壞遊戲平衡而不進行回報。
* 其他在公開社群中被合理視為不適當的行為。

## 執行與管理權限

作為專案維護者（開發者），我們擁有社群與專案的所有管理權限。我們有權利且有責任刪除、編輯或拒絕違反本行為準則的留言、GitHub Issues、Discord 訊息及其他貢獻。

若參與者出現不當行為，管理員有權在不事先通知的情況下，對其採取暫時或永久封鎖的措施（包含移出 Discord 伺服器或 QQ 群）。
