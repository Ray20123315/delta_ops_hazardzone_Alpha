# 貢獻指南

感謝您對 Delta Ops: Hazard Zone 的關注！您的參與是專案改進的關鍵。

## 參與方式

我們歡迎任何形式的貢獻，您可以依據您的興趣選擇參與路徑：

### 1. 報告錯誤 (Bug Report) - GitHub Issues
若您在遊玩過程中遇到崩潰、異常或 Bug，請至 [Issues](https://github.com/Ray20123315/delta_ops_hazardzone_Alpha/issues) 開啟新問題，並包含以下資訊：
- **問題描述**：發生了什麼事？
- **復現步驟**：如何重現此問題？
- **相關數據**：如 Crash Log、遊戲截圖或錄影。
- **環境資訊**：使用的 Minecraft 版本、Forge 版本及硬體概況。

### 2. 參與測試 (Testing & Feedback) - Q 群
我們非常重視測試人員的貢獻。如果您願意成為 Alpha/Beta 測試的一員：
- **加入討論**：請加入我們的 Q 群 (號碼: **714446429**)。
- **提供回饋**：在群內分享您的測試心得、效能數據或是功能建議。
- **貢獻記錄**：您的測試回報將被視為核心貢獻，參與測試且提供有效回饋者，將會被列入專案的 `CREDITS` 名單中，以感謝您的協助。

### 3. 交流與想法 (Discussions)
對於遊戲機制設計、未來功能展望，請前往 [Discussions](https://github.com/Ray20123315/delta_ops_hazardzone_Alpha/discussions) 參與交流。
